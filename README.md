# AI-vs-Real-Face-Classifier
A machine learning project that classifies AI-generated and real human faces using Teachable Machine.
# AI vs Real Face Classifier

This project was created using Google's Teachable Machine.

## Objective
Classify images as:
- AI Generated Face
- Real Human Face

## Dataset
- AI Faces: 995 images
- Real Faces: 995 images
